# FreelanceFlow 💰
### Pakistani Freelancer Income Dashboard

> Every dollar you earn. Every rupee you keep.

**[🔗 Live Demo](https://freelanceflow-dusky.vercel.app/)**

---

## The Problem

Pakistani freelancers earn in USD but think in PKR. After 
Fiverr's 20% cut, Payoneer fees, and exchange rate losses — 
they have no idea what actually hit their account.

**FreelanceFlow fixes this.**

---

## Features

- 💱 **Live Currency Rates** — real-time USD/PKR, EUR/PKR, GBP/PKR via API
- 🧮 **Real Earnings Calculator** — gross → platform fee → withdrawal fee → PKR conversion → FBR tax → actual take-home
- 🏦 **FBR Tax Calculator** — 2024-25 Pakistani freelancer tax slabs, accurate to the rupee
- 📊 **Income Dashboard** — 6-month bar chart, platform donut chart, goal tracker
- 🏆 **Platform Comparison** — Fiverr vs Upwork vs Freelancer vs Direct client
- 📝 **Income Logger** — log earnings, save to localStorage, export as CSV
- 🎯 **Goal Tracker** — animated SVG ring shows monthly PKR target progress
